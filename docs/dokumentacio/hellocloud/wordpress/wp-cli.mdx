# <FontAwesomeIcon icon="fa-solid fa-square-code text-success" /> WordPress CLI

Minden szerver automatikusan WordPress CLI -vel érkezik. Ez egy nagyon hasznos eszköz mind a WordPress weboldalt üzemeltetőknek, mind a fejlesztőknek egyaránt.

## Adatbáziskezelés

### WordPress adatbázis exportálása (mentése)

A mysqldump segédprogram futtatása a DB_HOST, DB_NAME, DB_USER és DB_PASSWORD adatbázis-hitelesítési adatokkal, amelyek a wp-config.php fájlban vannak megadva. Elfogad bármilyen érvényes mysqldump kapcsolót.

**Egyszerű mód: **

`wp db export` -> Ez generál egy SQL fájlt. Példul: domain-2023-05-08-191e58e.sql

**Használata:**

Belépsz SSH-n a szerververre konzolból/terminálból és a WordPress gyökérkönyvtárában kiadod a fenti parancsot.

További lehetőségek: https://developer.wordpress.org/cli/commands/db/export/


### WordPress adatbázis importálása (visszaállítása)

**Egyszerű mód: **

`wp db import domain-2023-05-08-191e58e.sql` -> Ez kiolvassa az adatbázis adatokat wp-config.php-ból (automatikusan) és **felülírja** a jelenlegi adatbázist a `domain-2023-05-08-191e58e.sql` tartalmával. 

**Használata:**

Belépsz SSH-n a szerververre konzolból/terminálból és a WordPress gyökérkönyvtárában kiadod a fenti parancsot.

További lehetőségek: https://developer.wordpress.org/cli/commands/db/import/

Ha még nem ismered a WP-CLI-t mindenképpen kezd el használni. Nem sok dolgot tanulni sem, hiszen pár parancsot kell használni. Fel fogod tenni magadnak a kérdést: "**Hogy élhettem eddig WP-CLI nélkül?!**"
