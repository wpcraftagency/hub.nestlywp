# <FontAwesomeIcon icon="fa-duotone fa-arrows-rotate text-primary" /> Bővítmények frissítése

A HelloPack-os bővítmények frissítése semmiben nem tér el a több WordPress bővítmény frissítésétől. A bővítmények frissítését a **WP Admin -> Bővítmények -> Telepített bővítmények** vagy **WP Admin -> Vezérlőpult -> Frissítések** menüpontok alatt teheted meg.
