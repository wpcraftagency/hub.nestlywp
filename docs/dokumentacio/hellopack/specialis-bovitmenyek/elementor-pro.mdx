# <FontAwesomeIcon icon="fa-duotone fa-link text-primary" /> Elementor PRO összekötése az Elementor.com fiókkal

**Az összekötés egyszerű:**

1. Lépj be a WordPress oldal adminjába, ahol össze szeretnéd kötni az oldalt az elementor.com-al
2. Kattints ide: Elementor -> Beállítások -> Jobb fent: Connect Account vagy My Elementor felirat.
3. Vidd végig a lépéseket, és kész is vagy.

### <i class="fa-duotone fa-badge-check text-success"></i> Ha minden rendben van, akkor ezt kell látnod:
