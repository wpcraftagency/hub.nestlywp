# <FontAwesomeIcon icon="fa-solid fa-distribute-spacing-vertical text-primary" /> Divi API kulcs aktiválása

**Pár egyszerű lépés és már aktív (frissíthető) lesz a Divi sablonod.**

1. Lépj be a **WordPress adminba**
2. Kattints a **Divi felirat**ra (bal lent)
3. Kattints a **Sablonbeállítások** linkre
4. Válaszd ki a **Updates – Frissítés** fület
5. Add meg a kapott **felhasználónevet** és API kulcsot
6. Kattints a **változtatások mentése** gombra
