# <FontAwesomeIcon icon="fa-solid fa-distribute-spacing-vertical text-primary" /> Régi Divi API kulcs felülírása egy új API kulccsal

Korábbi, például lejárt, vagy nem használt API kulcs felülírása.

## Lépések:

1. Lépj be a **WordPress adminba**
2. Kattints a **Divi felirat**ra (bal lent)
3. Kattints a **Sablonbeállítások** linkre
4. Válaszd ki a **Updates – Frissítés** fület
5. Töröld ki a korábbi Felhasználónevet és API kulcsot
6. **Változtatások mentése gomb** megnyomásával mentsd el.
7. Add meg az új Felhasználónevet és API kulcsot
8. **Változtatások mentése** gomb megnyomásával mentsd el.
