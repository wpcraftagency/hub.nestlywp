# <FontAwesomeIcon icon="fa-sharp fa-solid fa-microchip text-primary" /> Technikai adatok

**Szerverek helye:** Európában, Amerikában, Ázsiában és Ausztráliában. Mindig abba a szerverközpontba tesszük a weboldalt, amihez közelebb vannak az oldal látogatói.

**Háttértároló:** SSD

**Elérhető PHP verziók:** 7.4, 8.0, 8.1, 8.2 (PHP 7.4 alatti verziókat nem támogatjuk többé)

**SQL:** MariaDB, MySQL

**PHP CLI:** php-cli a fenti php verziókkal elérhető. Sőt, minden szerveren aktív WP-CLI-t is biztosítunk.

**Webszerver: **NGINX alapértelmezetten (aki .htaccess-t szeretne használni, annak rendelkezésére áll egy NGINX + APACHE hibrid megoldás).

**GIT:** Több GIT szolgáltatót is támogatunk. GitHub, GitLab, Bitbucket és egyedi GIT szerverek

**Szerveroldali cache rendszerek: **REDIS, Memcached, LiteSpeed
