# <FontAwesomeIcon icon="fa-duotone fa-code-compare text-primary" /> Visual Studio Code (VS Code) - A kódszerkesztő

A Visual Studio Code (VS Code) egy ingyenes és nyílt forrású kódszerkesztő, amelyet a Microsoft készített. A programot gyakran használják web- és felhőalapú alkalmazások fejlesztéséhez, de kényelmesen használható bármilyen nyelvhez vagy keretrendszerhez, beleértve a WordPress-t is.

Az VS Code rendelkezik olyan funkciókkal, mint a kódszínezés, automatikus javítás, Git-integráció, kiterjesztések és sok más, amelyek segítenek felgyorsítani a fejlesztési folyamatot. A felhasználók széles választékát szolgálja ki, beleértve a kezdő és haladó fejlesztőket is.

Az alkalmazás kialakítása modern, intuitív és könnyen használható. A felhasználóbarát felületen a különböző funkciókhoz könnyen hozzáférhetünk, ami javítja a produktivitást és a hatékonyságot. Az VS Code továbbá kiterjesztések által támogatja a testreszabhatóságot és az új funkciók hozzáadását.

Az alkalmazás keresztplatformos támogatást nyújt, ami azt jelenti, hogy használható Windows, macOS és Linux rendszereken is. Ezenkívül az VS Code-t a Microsoft folyamatosan frissíti és fejleszti, ami új funkciók és fejlesztői eszközök hozzáadását eredményezi.

Az Visual Studio Code egy hatékony és multifunkcionális kódszerkesztő, amely széleskörű lehetőségeket kínál a fejlesztők számára. Az ingyenesen letölthető és telepíthető, így érdemes kipróbálni, hogy láthasd, mennyire hasznos lehet a munkád során.

**+1 ha telepíted a magyar nyelvet, akkor a kódszerkesztő is magyarul tud.**

### Hasznos VS Code kiegészítők WordPress fejlesztéshez

1. **PHP Extension Pack **-- Tartalmazza a legfontosabb kiterjesztéseket, amelyekkel elkezdheted a PHP fejlesztést a Visual Studio Code-ban.

2. **Wordpress Snippet** (wpprotools.io) -- WordPress kódrészletek.

3. **ACF-Snippet** -- Speciális egyéni mező kódrészletek az Advanced Custom Fields-ből.

4. **WooCommerce** -- Kódrészletek és automatikus kiegészítések - Egy gyűjtemény WooCommerce kódrészletekből és automatikus kiegészítésekből.

5. **PHP DocBlocker** -- Egyszerű, függőségmentes, a PHP-hez készült DocBlocking csomag.

6. **phpcs** -- PHP CodeSniffer a Visual Studio Code-hoz.

7. **WPCS Whitelist Flags** -- WordPress Coding Standards (WPCS) engedélyezett jelzők.

8. **REST Client** -- A REST kliens lehetővé teszi, hogy HTTP kérést küldj és megtekintsd a választ közvetlenül a Visual Studio Code-ban.
