# <FontAwesomeIcon icon="fa-duotone fa-list-check text-primary" /> “cURL error 60: SSL certificate problem: certificate has expired” + Hotfix bővítmény

:::warning Figyelem
Ezt a megoldást csak akkor használd, ha az oldalad nem működik megfelelően a tárhelyed elavultsága miatt. Kérlek, csak addig használd, ameddig letisztázod a szolgáltatóddal a kérdést. Semmilyen módon nem ajánlott huzamosabb ideig használni ez a bővítményt, mert biztonsági kockázatokat hordozhat.
:::

Készítettünk bővítmény annak érdekében, hogy minden tárhelyen megfelelően működjenek az Let’s Encrypt frissítés miatt leállt bővítmények. A bővítmény egy filtert állít be, amelyik letiltja az HTTPS SSL tanúsítványok ellenőrzését a WordPress oldalakon.

**Innen lehet letölteni:** [Hotfix bővítmény letöltése](https://github.com/hellowpio/hub.hellowp.io/raw/main/docs/tudasbazis/hibaelharitas/assets/hellowp-ssl-problema-hotfix.zip)

Megold egy pillanatnyi hibát, amit a Let’s Encrypt frissítése okoz elavult szervereken.

**Telepítés:**

1. Töltsd le a bővítményt .zip formátumban.
2. Menj a WordPress admin felületére **"Bővítmények" -> "Új bővítmény telepítése"** oldalra, és kattints a **"Feltöltés"** gombra. Itt tudod feltölteni a letöltött .zip fájlt.
3. Miután feltöltötted a fájlt, kattints a "Telepítés most" gombra. A WordPress most telepíteni fogja a bővítményt, és megjelenik egy üzenet a sikeres telepítésről. Végül aktiválni kell a bővítményt a "Bővítmények" oldalon.
