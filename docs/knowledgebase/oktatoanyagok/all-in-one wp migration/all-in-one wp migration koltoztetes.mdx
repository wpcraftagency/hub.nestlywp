Ebben az útmutatóban megtudhatod, hogy hogyan költöztethetsz el egy oldalt az **All-in-One WP Migration** bővítménnyel. 

## 1. Előkészületek 

1. Győződj meg róla, hogy a **cél oldal tárhelyére** (ahová költöztetni szeretnéd az oldalt) feltelepítetted a **WordPress**-t. 
2. Telepítsd **mindkét** weboldalra (ahonnan költöztetsz, illetve ahová költöztetnél) az **All-in-One WP Migration** bővítményt, illetve az **All-in-One WP Migration Unlimited Extension** bővítményt. 
4. **Mindkét oldalon** a **Bővítmények** menüponton belül válaszd ki a **Telepített bővítmények** menüpontot.
5. Keresd meg az **All-in-One WP Migration Unlimited Extension** bővítményt, majd kattints a **Frissítések ellenőrzése** feliratra, majd frissítsd a bővítményeket. 
![AiO frissítés ellenőrzése](/img/all_in_one_wp_migration_frissites.png)

## 2. Weboldal mentése
A **költöztetendő** oldalon létre kell hoznunk egy mentést az **All-in-One WP Migration** bővítménnyel, ehhez kövesd a következő lépéseket:

1. A költöztetendő oldal vezérlőpultjának bal oldali sávjában vidd az egeret az **All-in-One WP Migration** menüpontra, majd kattints a **Mentések** menüpontra. 
2. Kattints a **Biztonsági mentés készítése** gombra. 
3. Amint készen van a biztonsági mentés, kattints a **Letöltés** gombra. 


## 3. Weboldal importálása

1. A **cél oldal** vezérlőpultjában vidd az egeret az **All-in-One WP Migration** menüpontra, majd válaszd az **Importálás** opciót. 
2. Kattints az **Importálás innen** gombra, majd válaszd ki a **Fájl** opciót. 
3. Válaszd ki a letöltött mentés fájlt. 
4. Az importálás végeztével a vezérlőpultban található **Beállítások** menüpontban válaszd ki az **Állandó hivatkozások** menüpontot.
5. Tekerj le az oldal aljára és kattints a **Módosítások mentése** gombra.
