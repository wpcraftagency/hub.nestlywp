# <FontAwesomeIcon icon="fa-brands fa-github text-primary" /> Hogyan töltsd fel a GitHubra a kódot?

Ha szeretnél új repository-t feltölteni GitHubra, akkor kövesd a következő lépéseket:

1. **Regisztrálj vagy jelentkezz be** a GitHub fiókodba.
2. Kattints a jobb felső sarokban található **"+" gombra**, majd válaszd ki az "Új Repository" lehetőséget.
3. **Adj meg egy nevet a Repository számára**. A névnek egyedinek kell lennie, és jól leírnia kell a projekted tartalmát.
4. Írj le egy rövid összefoglalót arról, hogy mi lesz a projekted célja. (opcionális)
5. Döntsd el, hogy publikus vagy privát lesz-e a repo. A publikus repositoryk bárki számára elérhetőek, a privát repositorykhoz azonban fizetős előfizetésre van szükség.
6. Válaszd ki a README fájl opciót. Ez egy alapvető fájl, amely lehetővé teszi, hogy bemutassad a projektet, és felvegyél vele kapcsolatos fontos információkat. (opcionális)
7. **Kattints a "Create Repository"** gombra.
8. **Ezzel elkészült a repository! **Most már hozzáadhatod a forráskódodat, dokumentációdat és egyéb fájlokat a projektjéhez. Ha szeretnéd, beállíthatod a repository beállításait, mint például az alkalmazások és szolgáltatások használatát, együttműködés hozzáadását és a webhookok beállítását.
