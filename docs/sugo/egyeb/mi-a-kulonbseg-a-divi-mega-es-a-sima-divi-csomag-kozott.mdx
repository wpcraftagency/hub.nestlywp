# Mi a különbség a Divi mega és a sima Divi csomag között?

A HelloWP két csomagban nyújt támogatást a DIVI-hez:

1. Prémium támogatás DIVI-hez és ajándék Divi License (Lifetime) – API kulcs - https://hellowp.io/hu/t/divi-license-lifetime/
2. DIVI MEGA CSOMAG + ajándék Divi License – API kulcs - https://hellowp.io/hu/t/divi-mega-csomag/

A két csomag között számos különbség van. Az első termék (Prémium támogatás) megvásárlásával **örökre szóló támogatást kapsz a Divi-hez **a VIP HelloWP Facebook csoportjában. Az ajándék API kulccsal pedig hozzáférsz minden tartalomhoz, amit a Divi alap esetben kínál (frissítés, alap sablonok, bővítmények, DIVI, EXTRA, BLOOM, MONARCH). Ez egyszeri összeg.

**Míg a DIVI MEGA CSOMAGBAN minden benne van az alapban is**, viszont adunk mellé több 100e Ft értékben kész weboldalakat, webshopokat, amit folyamatosan frissítünk, illetve új tartalmakat/oldalakat adunk hozzá a jövőben. Ezért nem korlátlan időre szól, hanem egy évre, hiszen **folyamatosan megújuló értéket kapsz**.

**A DIVI MEGA csomag tartalma:**

- Divi weboldal tervek, layout-ok: https://hellowp.io/hu/k/weboldal-tervek/
- Komplett DIVI oldalak: https://hellowp.io/hu/tema/divi-mega-weboldal/
- Komplett Divi webshopok: https://hellowp.io/hu/tema/divi-mega-webshop/
- Divi Szekciók: https://hellowp.io/hu/t/200-divi-szekcio-sablon/
- Divi fejlécek és láblécek: https://hellowp.io/hu/tema/divi-mega-fejlec-lablec/
