# <FontAwesomeIcon icon="fa-sharp fa-solid fa-credit-card text-primary" /> Manuális bankkártya terhelés

Amennyiben szeretnéd előfizetésedet megtartani, de valamilyen oknál fogva a rendszer nem tudta megterhelni a bankkártyádat annak összegével, úgy lehetőséged van manuális kifizetést indítani.

1. Nyisd meg a HelloConsole-t, vagy kattints a következő linkre: https://hellowp.io/hu/helloconsole/
2. Kattints a bal oldali sávban található "**Számlázás**" menüpontra, majd a "**Korábbi rendelések**" menüpontra. 

<BrowserWindow url="https://hellowp.io">
     <img
       src="/img/manualis-bankkartya-terheles-0.png"
       alt="manuális bankkártya terhelés"
     ></img>
   </BrowserWindow>

   3. Az aktuális, kifizetetlen rendelésnél kattints a "**Fizetés gombra**".
